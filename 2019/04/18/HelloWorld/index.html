MySQL 操作记录

发表于 2019-04-18

MySQL为关系型数据库，所谓的“关系型”可以简单的理解为“表格”的概念。

什么是关系型数据库

  • 关系型数据库是依据关系模型来创建的数据库。
  • 所谓的关系型模型就是“一对一，一对多，多对多”等关系模型，也就是指二维表格，因而一个关系型数据库就是由二维表及其之间的联系组成的一个数据组织。
  • 关系型数据可以很好地存储一些关系模型的数据。
  • 关系模型包括数据结构（数据存储的问题，二维表）、操作指令集合（SQL语句）、完整性约束（表内数据约束、表与表之间的约束）。

连接数据库

  • 通过ADO连接MySql数据库
  • 通过MySQL自己的API函数进行连接 [MySQL API][https://dev.mysql.com/doc/refman/5.7/en/c-api-function-overview.html]
  • mysql_real_connect()

    连接成功后返回MYSQL* 句柄.

  • mysql_query()

    执行指定的SQL语句

  • mysql_field_count()

    返回连接上最近查询的列数
